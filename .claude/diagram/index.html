📡 Full Request Lifecycle — Affiliate Campaign → Order Flow

STEP 1
Visitor clicks affiliate link → Browser navigates to site.com/john?campaign=tiktok1
MW
Next.js middleware.jsx intercepts — detects /john slug → resolves affiliate, sets cookie Set-Cookie: affiliate_id=aff_uuid_john; Max-Age=2592000 (30 days)
ROUTE
Next.js renders app/[username]/page.jsx → fires tracking ping POST /api/tracking/event { event: "PAGE_VIEW", source: "TIKTOK", landingPageId }
SVC
trackingService.js persists event, affiliateService.js increments click counter prisma.$transaction([trackingEvent.create(), affiliateClick.create(), affiliate.update({ totalClicks: {increment:1} })])
USER
User views product page, adds to cart, goes to checkout address form
ROUTE
Address submitted → checkout creates pending order via API POST /api/order { customerName, shippingAddress, products[], sessionId }
HOF
withAffiliate(handler) wrapper reads affiliate_id cookie → injects req.affiliateId into order req.affiliateId = "aff_uuid_john" ← read from HTTP cookie, attached to order
PRISMA
Atomic transaction: create Order with affiliateId + create OrderItems prisma.$transaction([order.create({ affiliateId }), orderItems.createMany()])
USER
User selects payment method (e.g. Stripe) → card form shown
ROUTE
Stripe PaymentIntent created via API route, confirmed client-side POST /api/payment/stripe → paymentService.createStripeIntent(amount, currency)
WEBHOOK
Stripe sends payment_intent.succeeded webhook to Next.js API route POST /api/payment/stripe/webhook → paymentService.verifyStripeWebhook()
PRISMA
Atomic transaction: update Order status + credit affiliate commission + record TrackingEvent prisma.$transaction([order.update({status:"CONFIRMED"}), affiliate.update({totalCommission:{increment:commission}}), trackingEvent.create({event:"ORDER_COMPLETED"})])
✅ DONE
Order confirmed · Affiliate commission recorded · Tracking analytics updated · Admin dashboard reflects live data

🌐 React UI ↔ Next.js API Routes

  • All data fetching via fetch('/api/...') (same origin — no CORS needed)
  • JWT stored as HttpOnly cookie, sent automatically on every request
  • Affiliate ID stored as affiliate_id cookie (30-day persistence)
  • Checkout forms POST to /api/order directly within Next.js
  • Admin pages call /api/admin/stats for dashboard metrics
  • Tracking pings fired as non-blocking fetch(..., { keepalive: true })

🔌 API Routes ↔ src/lib

  • Route handlers are thin — only call controller(req) and return response
  • Auth wrapping via withAuth(handler) HOF — no repeated auth code
  • Controllers validate input, call services, format apiResponse.success()
  • Services hold all business logic — fully testable without HTTP context
  • lib/prisma.js exports single PrismaClient shared across all modules

🔮 Prisma ↔ PostgreSQL

  • Connection via DATABASE_URL in .env file
  • Prisma generates optimised SQL from typed query builder calls
  • JSONB columns for flexible data: sections, shippingAddress, campaignData
  • All FK relations enforced at DB level — data integrity guaranteed
  • Migrations tracked in prisma/migrations/ — versioned schema changes
  • Indexes defined in schema.prisma → applied automatically via migrate dev
