# Project Rules

1. The project uses Node.js backend with Express.

2. Database is PostgreSQL using Prisma ORM.

3. The system contains:

* Products
* Orders
* Feedback system
* Affiliate system
* Landing page builder
* Checkout system
* Tracking system

4. Feedback system must support:

* text
* image
* audio
* scheduled publishing

5. Affiliate system must use URL format:

site.com/username

6. If user enters via affiliate link the system must keep the affiliate tracking active.

7. Landing page builder must support sections:

* hero
* video
* before_after
* gallery
* feedback
* faq
* cta

8. Admin panel must control:

* products
* orders
* feedback
* affiliates
* landing pages
* tracking
